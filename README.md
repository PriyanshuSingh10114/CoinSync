This is a simple web-based currency converter that allows users to convert between different currencies using real-time exchange rates fetched from an API.

Features
Convert currencies in real-time
Fetch latest exchange rates using an API
Simple and responsive user interface
Supports multiple currencies


Technologies Used
HTML
CSS
JavaScript
Exchange Rate API
